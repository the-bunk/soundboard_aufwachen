Click Count

Jedesmal, wenn du einen Sound abspielst, wird auf dem Server ein Zähler um eins erhöht, es werden keine weiteren Daten abgespeichert. Damit könne die beliebtesten Sounds ermittelt werden. So lange die Logdateien bestehen kann allerdings das Abspielen eines Sounds einer Uhrzeit und IP-Adresse zugeordnet werden.
